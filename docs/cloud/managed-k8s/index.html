    Overview

    The creation of a resource Cluster will trigger the deployment and provisionning of:

    • the Kubernetes cluster’s control plane, using OneInfra
    • the worker nodes using common Cloud APIs

    Example

    ---
apiVersion: datap.io/v1alpha1
kind: Cluster
metadata:
  name: my-cluster
  namespace: default
spec:
  controlPlane:
    replicas: 3
  workers:
    - cpu: 3GHz
      memory: 16GB
    - cpu: 3GHz
      memory: 16GB
    - cpu: 3GHz
      memory: 16GB

    NB: This example is only a draft.
